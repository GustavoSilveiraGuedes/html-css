Texto marcado

Podemos criar tambem um texto marcado usando a tag <mark> e </mark>.
Obs: (A tag <mark> e <mark> é semantica),
pois alem da forma dada tambem tem o sentido de querer que o texto tenha um distaque,
caso você queira trocar a cor do marca texto dai seria para CSS.
